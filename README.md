# HelloJSON

![JSON Icon](https://img.icons8.com/ios/50/000000/json.png)

Advanced JSON Viewer & Formatter – a lightweight, interactive web tool to paste, edit, format, and explore JSON data in a collapsible tree view. Perfect for developers, testers, or anyone working with JSON APIs.

---

## 🌟 Features

- **Paste / Copy / Load JSON** directly in the editor
- **Format / Minify** JSON with one click
- **Collapsible Tree View** for nested objects and arrays (like VS Code)
- **Expand / Collapse All** nodes instantly
- **Resizable Panels**: adjust editor & viewer width
- **History Sidebar**: stores last 50 JSON entries for quick reload
- **About Sidebar**: usage examples and developer info
- **Theme Support**: Light & Dark mode, persistent across sessions
- **Safe Parsing**: tries to recover incomplete or partially copied JSON
- **Auto-save**: remembers last JSON after refresh

---

## 💡 Use Cases

- Inspect API responses in a structured tree
- Debug JSON formatting issues
- Quickly explore nested data objects
- Educational tool to understand JSON structure

---

## 🖥️ How to Use

1. Paste your JSON in the left editor panel.
2. Click **Format** to pretty-print the JSON.
3. View the structured tree in the right panel.
4. Expand/Collapse nodes for better readability.
5. Use **Copy** to copy formatted JSON.
6. Access **History** to reload previous JSON entries.
7. Toggle **Theme** between light and dark modes.
